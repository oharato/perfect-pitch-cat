# 絶対音感トレーニング Webアプリ プロジェクト

## 1. 概要

ブラウザで動作する絶対音感トレーニング用のWebアプリケーションを開発します。
`Tone.js` ライブラリを使用してピアノの音を再生し、ユーザーが音を当てるゲーム形式のトレーニングを提供します。
